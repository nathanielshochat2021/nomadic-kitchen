## Phase 0 — Live (Week 0)

- [x] Site built and pushed to `main` → Vercel auto-deploys.
- [ ] Configure Resend env vars in Vercel (`RESEND_API_KEY`, `QUOTE_TO_EMAIL`,
      `QUOTE_FROM_EMAIL`) and send a test quote.
- [ ] Point a custom domain at the Vercel project.
- [ ] Replace placeholder prices in `lib/configurator.ts` with real costs.
- [ ] Swap in final photography (`public/images/`).

## Phase 1 — Foundation (Weeks 1–3)

- [ ] Finalize pricing & margins (see `docs/pricing.md`).
- [ ] Lock deposit terms & warranty language (see `docs/operations.md`); legal
      review.
- [ ] Set up business email, phone, and a CRM/inbox for quote requests.
- [ ] Stand up Instagram + Pinterest; publish 6–9 launch posts (build process +
      finished cart).
- [ ] Add basic analytics; define funnel events (configurator start, quote
      submit).
- [ ] Photograph 1–2 builds properly (hero + detail + lifestyle).

## Phase 2 — Demand (Weeks 4–8)

- [ ] Publish a "build film" (YouTube) and cut reels from it.
- [ ] Launch a small paid test: retarget configurator visitors on IG/Pinterest.
- [ ] Email auto-reply + 3-step nurture sequence (quote → deposit).
- [ ] Attend / table at 1 local event (market, overlanding meet, home show).
- [ ] Collect first deposits; document the build pipeline.

## Phase 3 — Momentum (Weeks 9–13)

- [ ] First deliveries → capture photos, testimonials, referral asks.
- [ ] Publish customer build stories.
- [ ] Review funnel metrics; double down on the best channel.
- [ ] Re-price/iterate options based on real attach data.
- [ ] Plan seasonal push (spring patio / tailgate season).
