# Images

The site references two photos:

- `hero-defender.jpg` — full-bleed hero (the cart on its trailer behind the Land Rover Defender).
- `cart-front.jpg` — feature / gallery shot (front view of the cart: cooktop, prep, fridge drawers).

Both files are already included in this folder. To swap in new photography,
**replace these two files with the same filenames** and the site will pick them
up — no code changes needed. The site builds either way.

Recommended: landscape orientation, at least 1600px on the long edge, optimized
JPG/WebP for fast loading.
